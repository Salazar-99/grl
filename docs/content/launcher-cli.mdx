## Commands

The launcher groups setup, verification, and run submission into command-line workflows.

## Validation

Configuration is validated before any cluster resources are touched.

```bash
grl launch --config launcher/example-config.yaml
```

## Secrets

Secrets are resolved from configured stores and passed only to the components that need them.

## Submission

Submission creates the run resources and records the generated run identifier.
