## OTel collector

Collectors receive metrics from cluster components, GPU exporters, and manually instrumented training code.

## ClickHouse

ClickHouse stores time-series data for high-cardinality run and actor dimensions.

```sql
SELECT run_id, avg(reward)
FROM grl_rewards
GROUP BY run_id;
```

## Grafana

Dashboards expose training progress, GPU health, and queue pressure.

## Alerts

Alerts should focus on stalled training, failed environment managers, and missing telemetry.
