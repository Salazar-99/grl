## Service

The service defines the lifecycle and tool execution methods used by rollout workers.

## Requests

Requests carry session identifiers, tool names, arguments, and timeout information.

```protobuf
message ExecuteToolRequest {
  string session_id = 1;
  string tool_name = 2;
}
```

## Responses

Responses include stdout, stderr, exit status, and structured error details.

## Errors

Errors distinguish user-command failures from manager, transport, and VM lifecycle failures.
